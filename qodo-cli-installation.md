### QODO All Platforms

```bash
# Install Qodo Command globally
npm install -g @qodo/command

# Verify installation
qodo --version

# Login to get started
qodo login
```

### COPILOT All Platforms

```bash
# Install Qodo Command globally
npm install -g @github/copilot

# Verify installation
copilot --version

# help
copilot --help
```
