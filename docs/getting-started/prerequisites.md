# Prerequisites

Before starting, make sure your environment meets the following requirements.

---

## 🖥 Operating System

- Windows 10 or later (Windows 11 recommended)

---

## 👤 User Permissions

- Standard user account (no administrator privileges required)
- Ability to:
  - create folders (e.g. `C:\sw\`)
  - modify user environment variables
  - run `.exe` and `.zip` files

---

## 🌐 Internet Access

- Required to download tools and packages
- Some corporate networks may restrict downloads—use approved sources if needed

---

- Opening Command Prompt or PowerShell
- Running basic commands:
